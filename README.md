## lcutronbun
卒業論文などの表紙を書くためだけのパッケージです．あまり利用価値はないかもしれない．

### 使い方

#### 読込み
プレアンブルで

 \usepackage{lcutronbun}

とします．特にオプションはありません．

#### 設定項目
卒業論文フォーマット用にいくつか命令を定義しています．
- \title{} 題目を指定
-- \title{電気系の系譜}
